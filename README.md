
# CRHI NGO Website

## Install
npm install

## Run Development
npm run dev

## Production Build
npm run build

## Deploy to Vercel
Push to GitHub then import into Vercel.

## Add Images
Put your images inside:

public/images/

Required:
- crhi-logo.png
- crhi-hero.jpg
- crhi-community.jpg
- crhi-gallery-1.jpg
- crhi-gallery-2.jpg
- crhi-gallery-3.jpg
- crhi-gallery-4.jpg
